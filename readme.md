# ALU Regex Data Extraction

## Overview
This program extracts useful data from raw text using regular expressions.

## Data Types
- Emails (with ALU validation)
- Credit cards (masked)
- URLs
- Phone numbers

## How to Run
1. Go to src folder:
   cd src

2. Run:
   python main.py

## Output
Results are saved in:
output/sample-output.json

## Notes
- Only valid ALU emails are categorized
- Credit card numbers are masked for security
